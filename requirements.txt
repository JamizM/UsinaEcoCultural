Baixar bootstrap: https://getbootstrap.com/
Baixar Node/React: https://nodejs.org/en

Para instalar React:
    cd frontend
    cd vite-project
    npm install

        Caso o bootstrap não esteja instalado (ou não esteja funcionando):
        npm i bootstrap@5.2.3

        Caso o MBD não esteja instalado (ou não esteja funcionando):
        npm install react-bootstrap bootstrap

        Caso o react-router-dom não esteja instalado (ou não esteja funcionando):
        npm install react-router-dom

        Caso o RevoCalendar não esteja instalado (ou não esteja funcionando):
        npm i revo-calendar (Este comando deve ser executado na pasta .//frontend)

    npm run dev
